##echarts
by 三土家的宇宁

　　这是个百度大总管整出来的用图表展示数据的插件，要用专业一点提高逼格的叫法就叫”数据可视化”，能生成各种柱形图，饼形图，折线图，地图…看着很酷炫碉堡，还能兼容浏览器。

　　附上官网的地址http://echarts.baidu.com/上面有各种示例和参考文档，项目里用的差得不多的话可以直接拿例子来改改。

　　echarts用的时候是要依赖百度和另一个绘图的库zrender的，所以如果想用ＡＭＤ方式引入echarts的话就得先require一下zrender；

　　之后就可以引入echarts库了，要是要画和图表种类多的话完全可以直接引个dist/echarts-all.js(经过压缩)，或者source/echarts-all.js(没有压缩)这里面有所有种类的图表，就可以，或者更好的就是按需引入，比如只用画个地图，就只引个echarts-map.js来用。

　　之后就来把图整出来了，有三步要做的：初始化echarts对象；设置option；在页面上渲染出来；

　　在页面里面先留好具有宽和高的div,我们就用它绘制图表,获取到页面里的这个div元素，

　　　　《div id=”chart” data-role=”chart” style=”width:100%;height:500px”》《/div》 //就用这个div了

　　　　var myEchart = window.echarts.init(config.$chart[0]);//用init方法，传这个div进去初始化得到对象，将来就用这个echarts对象渲染出图来；

　　然后就要设置option了，这个option里就是具体的图表要显示的东西，要怎么个样式来显示…这些，比如说画个有统计数据的信息的柱形图(http://echarts.baidu.com/doc/example.html在这里可以一个个找来改改调调看):

option = {
tooltip : {　　　　　　//设置鼠标停在图上时的提示
    trigger: 'axis',
    axisPointer : {            // 坐标轴指示器，坐标轴触发有效
        type : 'shadow'        // 默认为直线，可选为：'line' | 'shadow'
    }
},
legend: {　　　　　　//设图例的内容样式(这个就只设了下内容)
    data:['直接访问', '邮件营销','联盟广告','视频广告','搜索引擎']
},
toolbox: {　　　　　//设置对这个图操作的各种小工具
    show : true,
    feature : {
        mark : {show: true},
        dataView : {show: true, readOnly: false},
        magicType : {show: true, type: ['line', 'bar', 'stack', 'tiled']},
        restore : {show: true},
        saveAsImage : {show: true}
    }
},
calculable : true,　　//可以拖拽着图表的一个个区域来重新计算
xAxis : [　　　　　　//设置x轴
    {
        type : 'value'
    }
],
yAxis : [　　　　　　//设置y轴
    {
        type : 'category',
        data : ['周一','周二','周三','周四','周五','周六','周日']
    }
],
series : [　　　　　//设置每一项的具体数据和展示样式
    {
        name:'直接访问',
        type:'bar',
        stack: '总量',
        itemStyle : { normal: {label : {show: true, position: 'insideRight'}}},
        data:[320, 302, 301, 334, 390, 330, 320]
    },
    {
        name:'邮件营销',
        type:'bar',
        stack: '总量',
        itemStyle : { normal: {label : {show: true, position: 'insideRight'}}},
        data:[120, 132, 101, 134, 90, 230, 210]
    },
    {
        name:'联盟广告',
        type:'bar',
        stack: '总量',
        itemStyle : { normal: {label : {show: true, position: 'insideRight'}}},
        data:[220, 182, 191, 234, 290, 330, 310]
    },
    {
        name:'视频广告',
        type:'bar',
        stack: '总量',
        itemStyle : { normal: {label : {show: true, position: 'insideRight'}}},
        data:[150, 212, 201, 154, 190, 330, 410]
    },
    {
        name:'搜索引擎',
        type:'bar',
        stack: '总量',
        itemStyle : { normal: {label : {show: true, position: 'insideRight'}}},
        data:[820, 832, 901, 934, 1290, 1330, 1320]
    }
]

};

　　通过option就把关于图表要展示的信息和展现的样式设置好；

　　之后就是最后一步渲染页面了，很容易的，

　　myEchart.setOption(this.option);　　//通过之前初始化出来的echarts对象，使用它的setOption方法
然后就可以看到图表展现在页面上了;

再列上点比较常用的和比较坑爹的option的设置，想起来再加，再想再加…

给itemstyle里设柱和颜色:可以用个函数

   color: function(params) {
    // build a color map as your need.
    var colorList = [　　　　　//柱子们会循环加上colorlist里面的颜色
      '#C1232B','#B5C334','#FCCE10','#E87C25','#27727B',
       '#FE8463','#9BCA63','#FAD860','#F3A43B','#60C0DD',
       '#D7504B','#C6E579','#F4E001','#F0805A','#26C0C0'
    ];
    return colorList[params.dataIndex]
},<br/>

label里的formatter:a,b,c三个选项,分别代表图表的name属性值，x轴坐标值，单个柱的数据，中间可以加像斜杠，百分号，这些，再用设好position就可以显示在柱上了；

markpoint,symbol,可以给图上加点啊,图啊,各种小提示这些；
